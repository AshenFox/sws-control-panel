# SWS Control Panel

SPA приложение котрольной панели.

## Как посмотерть проект локально

- git clone https://github.com/AshenFox/sws-control-panel.git
- cd ./sws-control-panel
- npm install
- npm run start

## Технологии

- TypeScript
- React
- Redux
- ReduxThunk
- ReactRouter
- SCSS
- HTML
- SVG
